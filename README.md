Duration parsing support:
- ISO-8601 Duration support
- Simple Units Style

ISO-8601 Duration support:
PT15M         → 15 minuten  
PT1H30M       → 1 uur 30 minuten  
P2DT3H4M      → 2 dagen, 3 uur, 4 minuten


Simple Units Style:
1ns   → nanoseconds  
1us   → microseconds  
1ms   → milliseconds  
1s    → seconds  
1m    → minutes  
1h    → hours  
1d    → days
